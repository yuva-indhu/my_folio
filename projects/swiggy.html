Swiggy Food Ordering System

Description

Java based console application that simulates an online food ordering system similar to Swiggy. Users can view menu, select food items, place orders, and generate bill.

Tech Stack

  • Java
  • OOP Concepts
  • Collections
  • Scanner

System Flowchart

Flow Explanation

  1. Application starts.
  2. Food menu is displayed.
  3. User selects items and quantity.
  4. Total bill is calculated.
  5. Order confirmation is shown.
  6. User exits application.
