Nutrition Expert AI

Description

Nutrition Expert AI is an AI-powered web application that generates personalized diet plans, calorie estimates, and wellness recommendations based on user inputs using Large Language Models.

Tech Stack

  • Python
  • Streamlit
  • LangChain
  • Ollama / LLM

System Flowchart

Flow Explanation

  1. User enters personal and health details.
  2. Inputs are sent to AI model.
  3. AI generates diet plan and calorie estimation.
  4. Results displayed in dashboard.
  5. User can regenerate plans.
