RAG Question Answering System

Description

A Retrieval Augmented Generation (RAG) based system that allows users to upload documents and ask questions. The system retrieves relevant content from documents and generates accurate answers using LLMs.

Tech Stack

  • Python
  • LangChain
  • Vector Database
  • LLM (Ollama / OpenAI)
  • Streamlit

System Flowchart

Flow Explanation

  1. User uploads document.
  2. Text is converted to embeddings.
  3. Embeddings stored in vector database.
  4. User enters question.
  5. Relevant chunks retrieved.
  6. LLM generates answer.
