Library Management System

Description

Java based console application to manage books, members, issue and return operations using object-oriented programming and file handling.

Tech Stack

  • Java
  • OOP
  • File Handling
  • Collections
  • Scanner

System Flowchart

Flow Explanation

  1. System loads existing data from file.
  2. User menu is displayed.
  3. User selects operation.
  4. Invalid choice shows error message.
  5. Valid choice performs operation.
  6. Data saved back to file.
  7. User exits program.
